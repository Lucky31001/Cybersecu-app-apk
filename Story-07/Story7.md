Rebuild un APK
----------
Pour rebuild un apk nous avons utiliser la commande suivante de l'outil apktool:

```apktool b .\app\ -v -o apk_rebuild.apk > build.log 2>&1```

Cette commande nous permet de rebuild l'apk et d'obtenir un log de ce build contenant aussi les erreurs potentielles.

Ces deux fichiers sont disponibles dans le dossier Story-07
